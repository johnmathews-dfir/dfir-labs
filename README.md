# DFIR Labs

Hands-on digital forensics and incident response labs, focusing on:

- forensic acquisition
- evidence integrity
- filesystem analysis
- data recovery

## Labs

### DFIR-001 Removable Media Investigation
Simulated forensic acquisition and analysis of a USB device, including:
- imaging
- hashing
- integrity verification
- deleted file recovery

### DFIR-002 Deleted File Recovery
Work in progress – focused on recovery of deleted artefacts from forensic images.

This repository demonstrates structured, repeatable DFIR workflows using open-source tools.
